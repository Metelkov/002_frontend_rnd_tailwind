случайный генератор всего use Tailwind framework

RND нескольких видов, разные группы - разные rnd. Генерит из интервала от мин до макс, генерит случайный выбор из массива, генерит выбор из двух символов, генерит выбор из числа - переводит в строку и если длина =1 - вперед добавляет ноль и получается 08 например, если длина = 2 - ничего не добавляет.

WE596.(?)*1.(?)*2-68(?)*300(?)*4(?)*5-(??)*6

*1 = модификация первая, выбор из массива:

-без индекса = генерит пробел;

1 -индекс = 1;

2 -индекс = 2;

3 -индекс = 3;

4 -индекс = 4;

7 -индекс = 7, да, да именно 7, 5 и 6 не учавствуют

*2 = модификация вторая:

-без индекса = генерит пробел;

P - генерит букву P

*3 = модификация третья, выбор из 3 чисел:

3 - индекс = 3;

4 - индекс = 4;

5 - индекс = 5

*4 = модификация четвертая - просто числа:

1 - индекс = 1

2 - индекс = 2

3 - индекс = 3

4 - индекс = 4

5 - индекс = 5

6 - индекс = 6

*5 = модификация пятая: 0 или 1

0 - индекс = 0

1 - индекс = 1

*6 = модификация с проверкой длины строки:

?? - если длина один символ - приставляем в начало ноль.
